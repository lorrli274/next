Node can be run on Windows, macOS, many "flavours" of Linux, Docker, etc. (there is a full list on the nodejs [Downloads](https://nodejs.org/en/download/) page). Almost any personal computer should have the necessary performance to run Node during development. _Express_ is run in a _Node_ environment, and hence can run on any platform that runs _Node_.

In this course we provide setup instructions for Windows, macOS, and Ubuntu Linux.
